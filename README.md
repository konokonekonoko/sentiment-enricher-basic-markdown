# Sentiment Enricher: Basic Markdown
This module extends the [Sentiment TTRPG System](https://github.com/tomlaflin/sentiment)'s built in Text Enricher capabilities, allowing it to apply basic MarkDown Highlighting.

## Installation
### Foundry's "Install Module" Dialog
Paste this link into the "Install Module" Dialog's "Manifest URL" field and click "Install":<br>
`https://raw.githubusercontent.com/konokonekonoko/sentiment-enricher-basic-markdown/main/module.json`

Then enable the module in your world.

### Manual
#### Download
[Download this repo as a zip](https://github.com/konokonekonoko/sentiment-enricher-basic-markdown/archive/refs/heads/main.zip) and unzip it in your `FoundryVTT/Data/modules` directory. Usually `%user%\AppData\Local\FoundryVTT\Data\modules` on Windows or `/home/$USER/.local/share/FoundryVTT/Data/modules` on Linux.

The directory structure should be as follows:
```
FoundryVTT/
└ Data/
  ├ modules/
  │ ├ sentiment-enricher-basic-markdown-main/
  │ └ [other module directories]/
  └ [other directories]/
```
Then enable the module in your world.
